# Bienvenido a Foslog v0.2.0

Estamos encantados de anunciar el lanzamiento de **Foslog v0.2.0**, una actualización importante que sienta las bases para una plataforma más robusta y fácil de usar. Esta versión introduce mejoras significativas en la experiencia de usuario, internacionalización y autenticación.

## Novedades

### Experiencia de Usuario Mejorada
- **Paleta Crema Cohesiva**: Un nuevo tema claro con una paleta de colores crema cohesiva para una experiencia de visualización más agradable.
- **Experiencia Móvil Mejorada**: Hemos solucionado el zoom del navegador móvil al enfocar un campo de entrada utilizando un tamaño de fuente de 16px, lo que hace que la interfaz sea más amigable en dispositivos móviles.
- **Año Dinámico**: La función "fromDateOnwards" ahora utiliza un año dinámico, asegurando que la información esté siempre actualizada.

### Internacionalización
- **Selector de Idioma**: Ahora puedes cambiar entre inglés, español y catalán.
- **Preferencia de Idioma**: Tu preferencia de idioma se guarda en el almacenamiento local para una experiencia sin interrupciones.
- **Traducciones Faltantes**: Hemos añadido las traducciones que faltaban para proporcionar una experiencia multilingüe más consistente.

### Autenticación y Seguridad
- **Inicio de Sesión y Registro**: Hemos implementado la funcionalidad de inicio de sesión y registro, permitiéndote crear una cuenta y acceder a funciones personalizadas.
- **Integración con Better Auth**: Hemos integrado Better Auth para un sistema de autenticación más seguro y fiable.
- **Tabla de Verificación**: Se ha añadido una nueva tabla de verificación al esquema de Prisma para Better Auth.

### Bajo el Capó
- **Abstracción de Componentes**: Hemos abstraído componentes y añadido las pruebas que faltaban para mejorar la calidad y mantenibilidad del código.
- **Mejoras en CI/CD**: Hemos añadido un trabajo de Vitest a nuestro pipeline de CI para asegurar la calidad de nuestro código.
- **Actualización de Dependencias**: Hemos actualizado nuestras dependencias a las últimas versiones para mantener la plataforma segura y con buen rendimiento.

## Próximos Pasos

Esta versión es un paso adelante significativo para Foslog, pero apenas estamos empezando. Tenemos muchas características y mejoras emocionantes planeadas para el futuro, incluyendo:

- Un nuevo blog para compartir actualizaciones e ideas
- Una biblioteca de medios más completa
- Capacidades avanzadas de búsqueda y filtrado

¡Mantente atento a más actualizaciones y gracias por ser parte de la comunidad de Foslog!
